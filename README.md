# Programming Game

Projeto de Trabalho de Conclusão de Curso (TCC) desenvolvido em GameMaker.

## Sobre o projeto
O Programming Game é um jogo educacional criado com o objetivo de auxiliar no ensino de lógica de programação e conceitos básicos de Python de forma interativa.

## Objetivo
Proporcionar uma experiência de aprendizado mais dinâmica, utilizando elementos de jogos para reforçar conteúdos introdutórios de programação.

## Status do projeto
Em desenvolvimento

## Tecnologias utilizadas
- GameMaker
- GML (GameMaker Language)
